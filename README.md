# game-basics

A simple choose-your-own-adventure game built in Python with background art, character sprites, sounds, and a text box UI.

## Run the game

1. Open a terminal in this folder.
2. Run:

```bash
python choose_your_adventure.py
```

## Notes

- The game auto-generates simple artwork and sound files in `assets/` the first time it runs.
- Use the buttons at the bottom to choose your adventure path.
